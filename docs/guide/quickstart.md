# 你的第一個辦公室

這份帶你走完一次：**從打開控制台，到看著第一件事被做完**。全程一條路、不繞路，照著做大約十分鐘。
還沒裝好的話，先看 [安裝、升級與移除](install.md)；這裡假設安裝腳本已經跑完、印出了那行設定連結。

> 開箱時整間工作室只有兩個人：**你（owner），和 Mira**（你的助理）。這一趟就是你跟 Mira 的第一次共事。

---

## 第一步：打開控制台，設一組 owner 密碼

安裝完成時，畫面上會印出一行**一次性設定連結**，長這樣：

```
http://127.0.0.1:7755/?code=…
```

在這台機器上用瀏覽器打開它，設定你的 owner 密碼。這組密碼就是之後你登入控制台的憑證。

> - 這是**一次性**的：密碼一設好，那個 `code` 立刻失效，重裝也不會再印。
> - 之後要進控制台，直接開 <http://127.0.0.1:7755> 登入即可。
> - 密碼只以雜湊存進本機資料庫，不落在設定檔裡。
> - server **只綁 loopback（`127.0.0.1`）**——預設不對外，你的資料不會因為裝了它就上網。想從手機或別台機器連，得自己開一條通道，見 [在手機上用控制台](mobile.md)。

密碼設好後，server 會**自己接手最後兩步**：把這台機器的 warden 裝好、把預設助理 **Mira** 叫醒。
你不用自己去按安裝，也不用自己把助理設成上線。（缺 `claude` 或 `tmux` 時它會在控制台上方橫幅明確說明原因——那時請回頭看 [install.md](install.md) 的前置需求。）

---

## 第二步：認識 Mira

登入後你會落在 **Office（辦公室）** 分頁。左側是成員名單，頂部分成 **Staff（正職）** 與 **Outsource（外包）** 兩欄；
Staff 欄裡就站著 **Mira**。她名字旁邊有一顆 **presence 圓點**——顏色代表她在不在線：

| 圓點 | 意思 |
| --- | --- |
| Waking（喚醒中） | 剛被叫醒、還在開機 |
| Online（線上） | 已就緒，可以對話 |
| Offline（離線） | 沒在跑 |

第一次進來，Mira 通常正從 Waking 轉成 Online。等她亮成 Online，就可以開始了。

> Mira 是你的助理：溫暖、簡潔、務實。她維運這間工作室的日常，也接你交辦的雜事。
> 她底下就是一個跑在 tmux 裡的 Claude Code session——**跑的就是你機器上那個 `claude`**，你原本串好的 skill / plugin / MCP 原封不動全都在。

---

## 第三步：丟第一則訊息

點 Mira，右邊會開啟跟她的聊天。丟一句話試水溫，例如：

> 你好，先自我介紹一下，順便告訴我這間工作室現在有誰。

送出。你會看到她**先回一聲**（ack），再把答案講給你聽。幾件事現在就值得記住：

- **你打的字，只有透過控制台聊天送出去她才看得到。** 她看不到你的終端機，你也看不到她的——你們唯一的窗口就是這個聊天框。
- **她回話會很精簡。** 這是設計：一兩句講重點，不是丟一份報告給你。要她展開再叫她展開。
- **可以傳圖。** 聊天框能上傳截圖、mockup、設計稿；她也能把做好的成品回傳給你預覽。

---

## 第四步：交出第一件事，看它變成一張任務

現在交一件**多步驟、值得追蹤**的事——不是「幾點了」這種一句話回掉的小事。例如：

> 幫我把 `~/some-repo` 的 README 從頭讀一遍，列出三個最該修的問題，各附「在哪、為什麼、建議怎麼改」。

Mira 判斷這件事夠份量，就會把它**變成一張任務卡**，而不是在聊天裡零散做掉。到 **Task（任務）** 分頁，你會看到這張卡：

- 它被**拆成多個節點（step）**，每個節點寫明「**怎樣算做完**」（DoD，完成定義）。
- 卡上隨時看得到：**現在在第幾節、為什麼卡、卡了多久。**
- 做完一節，她**回報一節**，進度即時往前走。

這一刀是 OffiCraft 的核心：你交出去的不是一個一次性指令，是一條**帶完成定義的工作流**。
想深入了解任務怎麼運作、DoD 怎麼寫，看 [任務是怎麼運作的](tasks.md)。

---

## 第五步：答一張「等我回覆卡」

推進到某一步，如果有件事**只有你能拍板**（要不要動手、選哪個方向、能不能花這筆），Mira 不會替你決定——
她會**停下來，開一張「等我回覆卡」**，放進 **Ask（請示）** 分頁。Ask 分頁的徽章數字，就是還在等你回的卡數。

打開那張卡，你會看到：

- **一個問題**（一張卡只問一件事），寫得白話、一眼看懂。
- **背景與取捨**，夠你直接判斷——她會先把查得到的事實查好，不把功課丟回給你。
- **最多 4 個快捷選項**，第一個標著「AI 建議」，是她最推薦的做法。你也可以**直接打字回覆**，打字一樣算回答。

回一句，卡就關閉，任務**自動退回「進行中」**，Mira 接著把剩下的做完。

> 這就是 OffiCraft 對你的承諾：**該你決定的才給你，其餘她自己扛。** 你不需要盯著螢幕，卡會在那裡等你。

---

## 第六步：看它完成

Mira 把最後一個節點做完、所有 DoD 都達成後，任務**自動變成「已完成」**（你不用手動關）。她會把**交付物釘在任務卡上**——
PR 連結、報告、產出檔——你在卡上直接點得開，不必回頭翻聊天流。

到這裡，你已經走完一次完整的循環：

```
丟一句話 ─> 變成帶 DoD 的任務 ─> 需要你點頭時停下開卡 ─> 你回一句 ─> 做完、成果釘回卡上
```
