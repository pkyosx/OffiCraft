# 主題（外觀與用語）

主題決定控制台**長什麼樣、用什麼字**：一整套顏色，加上一份可選的用語覆蓋。內建只有一套**辦公室**（預設）；除此之外，你可以**自己新增主題**、調色、改用語，也能把主題**匯出成一個檔案**分享、或**匯入**別人做好的主題。

> 主題不會動到版面位置，只換顏色與用語；也**不會蓋過語言設定**——主題和語言各自獨立，你可以「辦公室配色 + English」或「自訂配色 + 中文」任意搭。

---

## 兩個地方，分工不同

| 在哪 | 做什麼 |
| --- | --- |
| **頭像下拉 › 偏好設定** | **選**要用哪個主題（辦公室，或你已建立／匯入的自訂主題）＋語言＋改密碼。只是切換，不在這裡編輯。 |
| **⚙️ 齒輪 › 設定 › 主題** | **管理**主題：新增、編輯顏色與用語、匯入、匯出、刪除。 |

一句話：**偏好設定選主題，設定頁管理主題。**

---

## 新增一個自訂主題

到 **設定 › 主題**：

1. **新增** → 給主題取個名字（例如「深海」）。它會以辦公室為底建立一份可編輯的副本。
2. **改顏色** — 顏色照用途分組（主色、背景、文字、狀態…），每一項給的是**看得懂的名稱**（例如「強調色」而不是內部代號）配一個**視覺取色器**，點開挑色即可。
3. **改用語（可選）** — 一份可搜尋的清單，每一列是「**英文原詞 + 目前顯示的文字**」，你對著要改的那條填新字。沒改的維持原本的語言。
4. 存檔後回偏好設定選它套用，整套顏色（＋你改過的用語）立刻上身。

> 顏色只能從**開放的具名色槽**裡挑（大約 40 個顏色角色），用語也只能覆蓋**既有的介面字詞**——你不能塞入任意 CSS 或程式碼。這是刻意的安全設計，避免破版與注入風險。

---

## 匯入與匯出（分享主題）

- **匯出** — 在設定 › 主題把目前主題輸出成一個 `.json` 主題包檔。自訂主題匯出的是**完整主題包**——顏色 + 用語 + 字型 + 頭像，一項都不漏；內建的辦公室主題本身沒有這些覆蓋，匯出的就是它整組顏色。
- **匯入** — 選一個 `.json` 檔（或貼上內容），系統驗證通過後就加進你的主題清單，跟自己建的一視同仁。驗證會擋掉非法色值、未知色槽、任意 CSS、或超長內容——匯入來路不明的主題包是安全的。

一個主題包就是一份純資料（顏色 + 用語 + 字型 + 頭像），拿給別人匯入就能得到一模一樣的外觀。

---

## 「修仙」主題

OffiCraft 早期把「修仙」做成內建主題（把介面用語整套換成修仙講法）。現在它不再是內建——**內建只留辦公室**。修仙改成一份**可匯入的主題包**：想試就用上面的「匯入」把修仙主題包匯進來，效果一樣，但它跟任何自訂主題一樣可選、可改、可刪，不再是寫死在程式裡的特例。
